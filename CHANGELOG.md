## 0.2.4-wip

- Add initial, basic support for `abnf` (Augmented Backus-Naur Form).
- Add initial, basic support for `bnf` (Backus-Naur Form).
- Add initial, basic support for `ebnf` (Extended Backus-Naur Form).
- Add a `Tags.grammarSource` tag to be used as
  the parent tag for grammar and syntactic metalanguages.

## 0.2.3

- Add initial, basic support for `python`.
- Improve support for flow-style productions in YAML.

## 0.2.2

- Add initial, basic support for `ruby`.
- Add initial, basic support for `glsl` (OpenGL Shading Language).

## 0.2.1

- Mark private property identifiers as private instead of special in Dart.

## 0.2.0

- Add basic, initial support for `c` and `cpp` (C++).
- Fix group counting when processing `Matcher.capture`.

## 0.1.1

- Add pub topics to the package.
- Add a basic example to the README.

## 0.1.0

- Implement initial support for `groovy` and `objective-c`.
- Improve support for `dart`, `java`, `json`, `kotlin`, `swift`, and `yaml`.
- Require Dart SDK version 3.9.0 or higher.

## 0.0.1

- In-progress, experimental release for `package:opal` with support for
  parsing strings from textual programming languages and data formats to
  lists of tokens, often for syntax highlighting.
